Install

  1. Open Windows Terminal → Settings (Ctrl+,) → Open JSON file.
  2. Copy both entries from the schemes array of terminal-workbench.json into the schemes array of your settings.json.
  3. Copy both entries from the themes array into the top-level themes array, creating it if it does not exist.
  4. Activate the theme:
{
    "theme": "Terminal Workbench",
    "profiles": { "defaults": { "colorScheme": "Terminal Workbench" } }
}

Color schemes apply to open tabs immediately; the window theme applies after Windows Terminal is closed and reopened. For light mode, use Terminal Workbench Light for both names.
